# UI-POLISH Runbook

## Objetivo
Mejorar claridad visual y experiencia sin romper funcionalidad.

## Criterios UX
- Jerarquía clara (primary/secondary/tertiary)
- Espaciado consistente
- Estados hover/active/focus coherentes
- Sin ruido visual innecesario

## Proceso
1. Captura estado actual (screenshot + problemas).
2. Priorizar 1-3 mejoras de mayor impacto.
3. Aplicar cambios visuales pequeños.
4. Validar en uso real.
5. Ajuste fino final.

## Checklist
- [ ] Se entiende en <1 segundo
- [ ] No overlap molesto
- [ ] No texto técnico visible
- [ ] Integración visual consistente
- [ ] Smoke/QA PASS
